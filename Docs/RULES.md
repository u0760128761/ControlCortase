# Правила работы AI Агента

1. **Комментарии в коде**: Все комментарии в коде должны быть написаны на **Русском языке**.
2. **Документация**: Вся документация должна храниться в папке `Docs` в корне проекта.
3. **Логирование действий**: Все действия по работе над проектом должны записываться в файл `Docs/CHANGELOG.md` в момент "коммита" (завершения задачи/этапа).
4. **Структура**: Поддерживать чистоту структуры проекта, новые файлы документации помещать только в `Docs`.
5. **Язык интерфейса**: Все элементы интерфейса должны быть на **трех языках**: Русский, Английский, Испанский. Язык интерфейса должен переключаться по кнопке на главном экране.
6. **Логика работы**: Приложение должно работать в фоновом режиме, сохранять состояние при выходе и восстанавливать его при возвращении. Приложение должно работать в фоновом режиме, сохранять состояние при выходе и восстанавливать его при возвращении.
7. **Документация**: Вся документация должна быть на **трех языках**: Русский, Английский, Испанский.
8. **Документация**: в интерфейсе приложения должна быть кнопка "Документация", которая открывает документацию на текущем языке интерфейса.
9. **Разработка**: Если ты переписываешь код, то старый код, который ты переписываешь, должен быть удален.
